# Sorting Algorithms

|             | Best Case | Worst Case | Average Case | Space Complexity |
|-------------|-----------|------------|--------------|------------------|
| Bubble Sort |   Ω(n)    |  O(n^2)    |  O(n^2)      |     O(1)         |
| Merge Sort  | Ω(n log n)| O(n log n) | O(n log n)   |     O(n)         |
| Quicksort   | Ω(n log n)| O(n log n) |  O(n^2)      |    O(log n)      |
